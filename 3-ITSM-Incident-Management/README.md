##  Scenario
The project team wants a solution to View and monitor a list of incidents and inquiries for the new VitaScope AR Field Device (VitaScope-FD). 

As a system administrator, I will create this using the List View feature in ServiceNow by customizing the incident table and creating a view.  

## Objective
Customized the incident table and Configuration Item table to create a view for monitoring incidents and inquiries for new device.

## Tasks Completed
- Reference the incident table and create a new colunm and include fields from Configuration table
- Create new view using list control
- View and monitor incidents and inquiry using classic view
- Create new application and module to hold new list for easy access
- Customize classic incident form (Adding new field, Service Offering and new Subcategories)
- Create new records on new reference table
- Enable auto-assignment of incidents using service offering
- Test incident assignment automation and view


## Result
A solution for the project team to view and manage incidents and inquires about new device.
