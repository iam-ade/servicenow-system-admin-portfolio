##  Scenario
Project team wants to ensure every technician has a chance to receive an Infinity device by ordering through a catalog item. Additionally, tracking orders and devices in one convenient location is appealing. 

## Objective
Create a service catalog item that technicians can use to order Vitascopre-FD device.

## Tasks Completed
- Create catalog admin group
- Add roles and members
- Create catalog item
- Configure variable and price
- Verify catalog item
- Confirm catalog item

A customized catalog itenm for technician to order Vitascoope-FD device.
