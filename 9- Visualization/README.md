## Scenario
Project team wants one place to go to get all the information for the Vitascope-FD product. 

## Objective
Create a dashboard to use for monitoring the service offering and create a visualization from a list to show Vitascope incident records using a filters.

## Tasks Completed 
- Create a dashboard to use for monitoring the service offering
- Create a visualization from a list to show HHD incident records by:
• Pilot vs. Non-Pilot
• Firmware vs. Non-Firmware


## Result
A customized dashboard with visualization showing information about incidents for pilot device
