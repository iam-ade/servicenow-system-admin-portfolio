# servicenow-system-admin-portfolio

## Overview
This project demonstrates hands-on ServiceNow system administration skills,
covering ITSM configuration, CMDB, automation, instance security, visualization, and reporting.

## Problem
Organizations struggle with inefficient IT service management due to
poor configuration, lack of automation, and limited visibility.

## Solution
A fully configured ServiceNow ITSM environment implementing:
- Structured user and role management
- Incident management
- Catalog Item
- Workflow Automation
- Data management
- Application and Access Control
- CMDB governance
- Knowledge Article Management
- Securing ServiceNow Instance
- Notification
- Operational dashboards Visualization

## Tools & Technologies
- Platform Overview
- Next Experience, Navigation, and Access
- Lists and Filters
- Forms
- Tasks
- Platform Analytics
- Knowledge Management
- Service Catalog
- ServiceNow ITSM
- CMDB
- ITIL v4 practices
