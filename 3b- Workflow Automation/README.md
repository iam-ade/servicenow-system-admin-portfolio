##  Scenario
Project team wants to ensure every technician has a chance to receive an Infinity device by ordering through a catalog item. Additionally, tracking orders and devices in one convenient location is appealing. 

## Objective
Implement a flow to facilitate delivery of the Vitascope-FD devices to technicians and ensure communication is provided to the requester.

## Tasks Completed
- Create workflow studio from scratch
- Define trigger and action to update a requested item record
- Add create catalog task action
- Send an email to the requester
- Activate and associate flow to service catalog item
- Verifiy Workflow

## Result
A customized worflow for ordering Vitascope-FD catalog item
