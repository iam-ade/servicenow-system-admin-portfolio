# User, Role & Group Management

## Objective
Configure structured user access using roles and assignment groups.

## Tasks Completed
- Created departments (IT, HR, Finance)
- Created end users and IT support users
- Configured assignment groups
- Assigned ITIL roles to support users

## Access Model
- End users: Incident creation only
- IT support: Incident management via ITIL role

## Result
A secure and organized user access structure aligned with best practices.
